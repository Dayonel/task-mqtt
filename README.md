## Installation
Install [Mosquitto](https://mosquitto.org/download/)
Add to env variable or navigate where installed

## Run project
```
npm run dev
```

## Add a migration
```
npm run generate
```

## Run supabase locally
```
supabase start
```

## Apply migration locally
```
supabase db reset
```

## Create a new function
```
supabase functions new hello-world
```
